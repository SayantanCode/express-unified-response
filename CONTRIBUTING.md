# Contributing to Express Unified Response

Thank you for your interest in contributing! 

## Setup Instructions
1. Fork the repository.
2. Clone your fork: `git clone https://github.com/SayantanCode/express-unified-response.git`
3. Install dependencies: `npm install`
4. Create a new branch: `git checkout -b feature/your-feature-name`

## Testing
Please add tests to your changes and ensure all tests pass before submitting a Pull Request:
```bash
npm test
```
## Pull Request Process
1. Ensure your code follows the existing style (run npm run lint).
2. Update the README or documentation if you changed the API.
3. Submit your PR against the `dev` branch.
